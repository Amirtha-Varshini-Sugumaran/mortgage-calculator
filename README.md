# Mortgage Calculator (Java)

A simple command-line mortgage calculator built using core Java concepts, demonstrating strong fundamentals in problem-solving, mathematical modeling, and clean code practices.

## Key Highlights
- Implemented mortgage calculation using financial formula and `Math.pow()`
- Applied core Java concepts: variables, constants, arithmetic operations, casting
- Handled user input using `Scanner`
- Formatted output using `NumberFormat` for real-world currency representation

## Tech Stack
- Java
- Core Java Libraries (`Math`, `Scanner`, `NumberFormat`)

## Functionality
- Accepts user input for:
  - Principal amount
  - Annual interest rate
  - Loan period (years)
- Calculates and displays monthly mortgage payment

This project demonstrates the ability to:
- Translate real-world financial logic into code
- Write structured, readable, and maintainable programs
- Apply foundational programming concepts accurately
