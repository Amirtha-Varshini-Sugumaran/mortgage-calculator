# Mortgage Calculator - Java

A simple command-line mortgage calculator built with core Java. The program takes loan inputs, applies the standard mortgage payment formula, and prints a formatted monthly payment.

## Why This Project Matters

This project shows the ability to translate a real-world financial calculation into clear, readable code. It is intentionally small, but it demonstrates input handling, numeric logic, formatting, and basic command-line program structure.

## Key Highlights

- Implemented mortgage calculation using financial formula and `Math.pow()`
- Applied core Java concepts: variables, constants, arithmetic operations, casting
- Handled user input using `Scanner`
- Formatted output using `NumberFormat` for real-world currency representation

## Tech Stack

- Java
- Core Java Libraries (`Math`, `Scanner`, `NumberFormat`)

## Functionality

- Accepts user input for:
  - Principal amount
  - Annual interest rate
  - Loan period (years)
- Calculates and displays monthly mortgage payment
